# Extrator de Vetores AIC para QGIS

Este projeto utiliza a API Gemini para converter prints de AIC em planilhas estruturadas.

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/SEU_USUARIO/NOME_DO_REPOSITORIO/blob/main/AIC_Extrator.ipynb)

### Instruções para o Operador:
1. Clique no botão acima para abrir o Colab.
2. Vá ao menu lateral (ícone de chave 🔑) e adicione seu `GEMINI_API_KEY`.
3. Execute as células de código e faça o upload dos prints da AIC.
4. O arquivo `.xlsx` gerado está pronto para importação no QGIS.
